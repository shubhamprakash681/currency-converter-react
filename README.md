## Installation & Setup

Clone the repository:

```bash
    git clone https://github.com/shubhamprakash681/currency-converter-react.git
```

Navigate to the project directory:

```bash
    cd currency-converter-react
```

Install dependencies:

```bash
  npm install
```

Run the development server:

```bash
  npm run dev
```

Open in browser:

```bash
  http://localhost:5173/
```
